Mac终端改造

发表于 2018-11-17 | 更新于: 2018-11-18 | 分类于 善事利器
字数统计: 623 字 | 阅读时长 ≈ 3 分钟

平时Mac使用的shell使bash, 它虽然能满足我们的需要, 但是使用起来不是很方便. zsh可配置性强, 用户可以自定义配置, 个性化强.

更换shell

  1. 查看当前使用的shell

    echo $SHELL

  2. 查看系统当前内置的shell

    cat /etc/shells

    如果没有发现/bin/zsh, 就需要手动安装Zsh.

  3. 更换shell

    chsh -s /bin/zsh

  4. 重启

    echo $SHELL

    替换成功!

安装oh-my-zsh

oh-my-zsh用来管理Zsh配置的.

  1. 安装

    sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)".

  2. 更改配置

    vim ~/.zshrc

    以上是部分配置, 需将#删除才能生效. 键入i进入编辑模式, 修改后esc退出编辑模式, 键入:wq保存退出. oh-my-zsh有支持的主题和插件, 可以自由配置.

  3. 插件推荐

    1. zsh-syntax-highlighting

      1. 安装

        git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

      2. 修改配置.

        vim ~/.zshrc

      3. 让修改生效.

        source ~/.zshrc

    2. zsh-autosuggestions

      1. 安装.

        git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

      2. 修改配置.

        vim ~/.zshrc

      3. 让修改生效.

        source ~/.zshrc

好了, 打完收工!
